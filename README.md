# DEA-fronteira-de-efici-ncia-para-os-cursos-de-ADS-usando-ENADE-2008-2021-
Avaliar a eficiência relativa dos cursos de Análise e Desenvolvimento de Sistemas (ADS) ao longo da série histórica ENADE 2008–2021 usando Análise Envoltória de Dados (DEA). Ou seja: comparar cursos (DMUs) e medir quem entrega maior resultado (desempenho dos alunos) com os recursos disponíveis.
